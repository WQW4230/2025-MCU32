# STM32 FunOS
一个基于 STM32F103C8T6 的多任务嵌入式系统，包含小游戏、红外遥控菜单、三轴机械臂、rtc 实时时钟等功能的小项目。

## 功能
- 通讯录、扫雷、贪吃蛇、猜数字小游戏
- 三轴机械臂控制（ADC+DMA+双摇杆）
- 可调速直流电机风扇，支持正反转
- RTC 实时时钟、闹钟
- 红外遥控菜单选择
- 可调速LED
- OLED 显示
